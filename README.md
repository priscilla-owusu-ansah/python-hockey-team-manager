# Hockey Team Manager

A Python desktop application for managing hockey teams.

The project demonstrates object-oriented programming and includes both a command-line interface and a graphical interface built with Tkinter.

## Features

- Register new teams
- Display teams in a table
- Delete teams
- View statistics
- Input validation
- GUI built with Tkinter
- CLI version included

## Run the GUI

```
python3 gui.py
```

## Run the CLI version

```
python3 main.py
```

## Technologies

- Python
- Tkinter
- Object-Oriented Programming
